Quick Help

Best demo sequence

1) Add a Country (name auto-generates flag if known).
2) Add a Course under that country (e.g. Turffontein, Greyville).
3) Add a Race Day (date).
4) Bulk-add races (count + start time + interval).
5) Add horses + Win/Place odds per horse.
6) Add Users + balances for the demo.
